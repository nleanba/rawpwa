Collatz Conjecture Calculator

The Collatz Conjecture states that you will eventually end up at 1 / end up un a 4 → 2 → 1 → 4 → 2... loop, if you take any natural number, and repeat the following step:
  • If the number is even, divide it by two.
  • If the number is odd, triple it and add one.
It could neither be proven right nor wrong yet. Wikipedia
